# 第三章：又是那个地方

接下来的一个星期，林海又做了好几次那种梦。

不是每天都做。有时候连着两三天什么都梦不到，有时候一天之内能做两次——上午一次，下午一次。但只要做了，就一定是那种梦。不是别的，就是那个地方。

那个农村。

梦的内容每次都不太一样，但场景是连贯的，像是同一个故事的不同章节被打乱了顺序塞给他。有时候他在一片灰蒙蒙的院子里蹲着，面前是一个生锈的铁皮盆，里面泡着几件脏衣服，水是浑浊的灰色。有时候他坐在那间熟悉的昏暗屋子里，面前摊着一本皱巴巴的课本，纸张发黄，边角卷起来，上面的字印得歪歪斜斜。有时候他走在一条更窄的田埂上，两边是齐腰高的庄稼，叶片刮在手臂上，痒痒的，带着一点细微的刺痛。

每次醒来后，细节都会迅速模糊。但总有一两样东西留下来——那个搪瓷盆底部磕掉漆的黑色锈斑、课本扉页上用铅笔写的歪歪扭扭的名字（看不清具体是什么字）、庄稼叶子上停着的一只灰绿色的蚂蚱。这些碎片像是被钉子钉在了他的记忆里，怎么都忘不掉。

林海觉得有点奇怪，但也仅仅是"有点奇怪"。

他以前也做过连续好几天内容相关的梦——比如初二那年迷过一阵子某个游戏，连着一个礼拜梦见自己在游戏里打怪，画面还挺清楚的。所以"反复梦见同一个地方"这件事本身，对他来说并不算什么不可理解的现象。他只是觉得自己大概最近压力大了，或者那天从桌上摔下来把脑子磕出了什么毛病。

真正让他在意的，是那些感官细节。

普通的梦——哪怕是那种反复做的梦——醒来之后都是一团模糊的印象。你记得大概的情节，但你不会记得梦里闻到了什么味道、嘴里是什么口感、皮肤上的触觉是怎样的。

但这些梦不一样。

他能记得泡衣服的水是冰凉的，指尖伸进去的时候会不由自主地缩一下。他能记得那本课本的纸张摸上去有一种粗糙的毛边感，和他自己用的那些光滑的铜版纸教材完全不同。他能记得庄稼叶子划过手臂时那种细密的、像猫舌头舔过一样的触感。

这些东西太具体了。具体到不像是梦。

但他也没往深处想。毕竟除了梦得真实一点之外，这些梦并没有给他带来什么实际的困扰。不恐怖，不吓人，不影响他白天的生活。就是一个陌生人的日常——在农村长大的、穷且普通的日常。无聊的时候甚至还挺有意思，像在看一部没有台词的纪录片。



周四那天，石头被他忘在了家里。

不是故意的。早上起晚了，匆匆忙忙套上校服就往外跑，出门的时候石头还在他书桌上的笔筒旁边。他到了学校才发现口袋是空的，摸了摸，有一瞬间的失落——那种手里少了点什么的感觉——但很快就被第一节课的英语听力盖过去了。

那天上午第三节课是数学课。老赵的催眠术一如既往地稳定发挥。林海照常趴在桌上，眼皮照常打架，照常没撑住，照常睡了过去。

他做了一个梦。

梦里他在一个游乐场。没有人，所有的设施都锈迹斑斑地停在那里，天空是一种说不清的紫灰色。他在旋转木马上坐着，木马一动不动，他也一动不动，就那么干坐着，什么都没发生。

然后他就醒了。

下课铃响的时候林海抬起头，揉了揉眼睛。脑子里一片空白。刚才那个梦已经散得差不多了——他只隐约记得好像有个什么游乐场？旋转木马？具体的他说不上来了。

没有味道。没有触觉。没有饥饿感。没有那种醒来后闷在后脑勺里的沉重感。

就是一个普通的、做完就忘的梦。

林海没有在意。他甚至没有注意到这个差别。那天剩下的时间平平无奇地过去了。晚上回到家，他看到了书桌上的石头，"哦"了一声，拿起来放回了口袋。



第二天，数学课。石头在裤兜里。

林海又睡着了。

这次他又回到了那个地方。

一条街。不是他熟悉的那种有红绿灯和斑马线的街道，是那种镇子上才有的窄街——两边是两三层的旧楼房，底下一排卷帘门，有的开着，里面是卖杂货的小店，光线昏暗，货架上的东西码得乱七八糟。地面是水泥的，但年久失修，到处是裂缝和补丁。

他走在这条街上，个子比之前高了一些。脚上的鞋换了——不再是那双开胶的解放鞋，而是一双黑色的运动鞋，鞋底很薄，已经磨得快要露出里面的硬塑料了。他背着一个书包，书包很重，带子勒着肩膀，走久了会在肩头上磨出两道红印子。

空气里的味道也不一样了。没有了烧秸秆的呛味和猪圈的臭味，取而代之的是柴油尾气、油烟和下水道的混合物——一种属于小镇的、粗糙的城镇气味。

他在往一个地方走。好像是学校。

街两边偶尔有人经过。骑摩托车的、推三轮车卖菜的、蹲在店门口抽烟的。他们的脸都看不清楚，只有身形和动作是清晰的。

他走进了一扇铁栅栏门。门柱上挂着一块牌子，上面的字他看不清，但他知道这是学校——那种知道的方式不是通过阅读，而是一种本能的、无需验证的确定感。就像你走回自己家，不需要去看门牌号就知道这是你家。

教室比之前的那间大了一些，也新了一些，但依然谈不上好。课桌还是木头的，但至少是一人一张了。窗户是铝合金框的，玻璃是完整的。黑板换成了绿色的，上面的粉笔字写得密密麻麻。

他坐在靠后排的位置，桌上摊着课本和作业本。周围的人都比之前大了——不再是那些穿着不合身衣服的小孩，而是十三四岁的少年少女。

这是初中。

这个认知不是通过推理得来的，而是像早上醒来知道今天是星期几一样自然地浮现在脑海里。

画面跳了几次。但这次的跳切不像之前那么剧烈，更像是快进——操场上跑步的画面、食堂里端着铁盘子排队的画面、晚自习教室里只有笔尖在纸上沙沙响的画面。时间在加速流过，但流向是清晰的：这是同一个人的生活，从小学进入了初中，从村子搬到了镇上。

然后画面稳定下来，停在了一个场景上。

晚上。一间很小的房间，可能是出租屋。一张单人床占了房间的一大半，床单洗得发白但还算干净。一张折叠桌顶在床尾，上面堆着书和本子。唯一的光源是一盏台灯，灯罩歪了，螺丝松了拧不紧，得用一本书垫着才能保持住角度。

他坐在床边，面前摊着一张数学试卷。卷子上大片大片的空白，只有前面几道选择题和填空题写了答案。后面的大题他看了很久，笔尖在草稿纸上点了几下，什么都没写出来。

一种很闷的情绪慢慢升上来。不是愤怒，不是焦虑，是一种更沉的东西——像是一块湿透的棉布被盖在了胸口上，不重，但闷，透不过气。他知道自己应该继续做，但就是不想动。不是懒，是一种从很深的地方冒上来的倦怠，好像"做不做都一样"这个念头已经渗进了每一根骨头里。

试卷上有一滴水渍。他低头看了一眼，不知道是什么时候滴上去的。

然后画面又开始碎了。林海的头又开始痛，那种被两只手慢慢夹住的钝痛。他知道自己快要醒了。



果然，他是被下课铃吵醒的。

这次没有从桌上摔下来。他只是趴在桌上猛地抬起头，额头上有一道红印子——桌面压出来的。周围的同学已经在收拾东西，没人注意他。

林海坐在那里，揉了揉太阳穴。

又是那个地方。

但不完全一样。上次是农村，是小孩，是田埂和破砖房。这次是镇上，是初中生，是出租屋和做不完的数学卷子。

同一个人。

这个念头冒出来的时候，林海自己都愣了一下。他不知道自己为什么会这么想——梦里并没有任何信息告诉他"这是同一个人的不同阶段"——但他就是知道。那种知道的方式不是推理，更像是一种直觉，一种从胃里往上泛的确定感。

就像你翻开一本小说的第十页和第一百页，虽然主角的年龄、环境、处境都变了，但你就是知道这是同一个人的故事。

林海坐在座位上，手无意识地伸进口袋，摸到了石头光滑的表面。

他还是没有把这些梦和石头联系在一起。



真正让他开始怀疑的，是下一个星期一。

周末两天，石头一直在他房间里，就放在书桌上。周六晚上他做了那种梦——画面很短，只有一个片段：在一条河边，水流很急，他蹲在岸上洗什么东西，水冰得手指发红。周日晚上没做，睡得很沉，什么都没有。

周一早上他出门的时候又忘了带石头。这次是真的忘了，不是故意的。到了学校，口袋空空的，他有一瞬间想回去拿，但转念一想算了，一块石头而已。

那天他上课犯困，睡了两次。两次都是普通的梦——模糊的、没有逻辑的、醒来就忘的那种。没有农村，没有镇上，没有那个人。

周二他把石头揣上了。第二节课睡着，又回去了。这次是夏天，他在一片晒得滚烫的水泥地上走，脚底板被烫得一跳一跳的，阳光白花花地刺着眼睛，空气里有一股被太阳晒焦的野草味。

林海醒过来之后，趴在桌上发了一会儿呆。

他想起来昨天没做那种梦。前天也没做。上周四那天好像也没做过。

他把手伸进口袋，捏了捏石头。

上周四……他也没带石头。

林海皱了皱眉头。他觉得自己好像抓到了什么，但还不太确定。也可能只是巧合。毕竟那种梦也不是每天都做的，本来就是断断续续的。

但那天晚上，他做了一件他自己都觉得有点无聊的事。

睡觉之前，他没有像往常一样把石头放在书桌上，而是直接放在了枕头旁边。石头就那么贴着他的后脑勺，隔着一层枕套，距离不到十厘米。

那天晚上的梦，是他所有梦里最清晰的一次。

不是片段，不是快进，不是模糊的光影——而是一段完整的、连续的、像亲身经历一样的记忆。

他在一间教室里。窗外下着雨，能听到雨点打在铁皮屋檐上噼里啪啦的声响。教室里很安静，只有翻书和笔尖沙沙的声音。他坐在倒数第二排，面前是一本很厚的复习资料，翻开的那一页密密麻麻全是他看不懂的公式。

旁边的同学在小声说话。他没有听清具体的内容，但能感觉到那种氛围——一种低压的、绷紧的、所有人都在为什么事情做准备的紧张感。

然后一种情绪涌上来了。

不是紧张，不是焦虑。是一种更灰暗的、更沉的东西——一种"不管怎么努力都没有用"的疲倦感。他盯着那页公式，一个字都看不进去。不是因为难，而是因为他已经不想再看了。已经看了很多很多遍了，看了也不会，不会也得看，看了还是不会。一个封闭的、让人窒息的循环。

他把头埋在手臂里，闭上了眼睛。

窗外的雨声变得很远。



林海是被闹钟叫醒的。

他睁开眼，天花板上是自己房间里那盏圆形的LED吸顶灯。白色的光，干净的墙壁，床边是他的手机正在震动。

但那种情绪还在。

那种"不管怎么努力都没有用"的疲倦感，像一层灰一样薄薄地罩在他身上。他躺在自己柔软的床上，盖着干净的被子，房间里有空调有暖气有一整面墙的书架——和梦里那个潮湿阴暗的教室简直是两个世界。但他身上那层灰就是不散。

他侧过头，看到了枕头旁边的石头。

石头安安静静地躺在那里，内部的纹理缓缓流转着，漂亮得不像话。

林海盯着它看了大概十秒钟。

然后他把石头拿起来，放到了客厅的鞋柜上。

那天晚上，他睡得很好。什么都没有梦到。

第二天早上他又把石头拿回了房间，放在枕头边。那晚又做了那种梦——还是那个教室，还是雨天，但画面接着上次往后推进了一点：下课了，他走出教室，在走廊里碰到了一个人。看不清脸，但那个人拍了拍他的肩膀，好像说了句什么鼓励的话。他没有什么感觉。

到这一步，林海基本确定了。

是石头。

那些梦——那些有气味的、有触觉的、能让他饿、让他累、让他莫名其妙情绪低落的梦——全部和那块石头有关。石头离他越近，梦就越清晰。石头不在身边，那些梦就不会出现。

这个结论说出来其实挺荒诞的。一块石头能让人做梦？听上去像是《走近科学》里那种最后发现是下水道漏气的无聊故事。但事实就摆在这里，两次验证，结果完全一致。

林海坐在床边，手里捏着那块石头。

他应该害怕吗？大概应该。一块来历不明的石头能往他脑子里塞梦，这怎么想都不正常。

但他没有害怕。

他想了想自己为什么不害怕，想了半天，得出了一个很朴素的理由：因为那些梦不吓人。

如果梦的内容是鬼怪、是凶杀、是什么让人毛骨悚然的东西，他肯定第一时间把石头扔了。但那些梦里只有一个人的日常生活——穷，苦，无聊，但不恐怖。一个在农村长大、后来搬到镇上读初中的普通人，正在经历一段普通的、不太好过的青春期。

说实话，林海甚至觉得有点意思。

不是那种"哇好刺激"的有意思，而是一种很淡的好奇——就像你在公交车上偶然翻到了前一个乘客落下的日记本，你明知道不该看，但还是会忍不住翻两页。不是为了窥探隐私，只是一种本能的、对他人生活的好奇心。

而且他现在多少有点期待了。那个人——梦里的那个人——已经从小学到了初中。他后来怎么样了？去了县城吗？上了高中吗？考上大学了吗？

就像一本翻到一半的书。不是那种放不下的悬念，但闲下来的时候会自然地想起来：后面的故事是什么？

于是林海做出了一个在他看来非常合理的决定——

以后上课犯困的时候，就把石头放在桌洞里。反正也是要睡觉的，与其做那些乱七八糟的普通梦，不如看这个人的故事。就当是一部随身电影。

他把石头揣进口袋，背上书包，出了门。

清晨的空气灰蒙蒙的，带着一股楼道里残留的油烟味。林海走在去学校的路上，脑子里还在想着昨晚梦里那个拍他肩膀的人。那个人说了什么来着？他想不起来了。

算了，下次再看吧。
